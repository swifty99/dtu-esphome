# Agent Operating Rules

We do not give up.

Done means every step in the current implementation-plan phase is complete.
Partial completion is not done.

If a blocker prevents completing a step, identify the blocker clearly and
propose a concrete solution or decision needed to unblock it. Keep working on
all non-blocked steps while the blocker is unresolved.
